Café Review API Information

Welcome to the Café Review API. Below, you'll find information about the available API endpoints that return JSON responses and examples of their output.

Endpoints

  • Random Café: /random

    Get information about a random café.

    {
    "cafe": {
        "id": 1,
        "name": "Example Café",
        "map_url": "https://example.com/map",
        "img_url": "https://example.com/image",
        "location": "Example Location",
        "seats": "20",
        "has_toilet": true,
        "has_wifi": true,
        "has_sockets": true,
        "can_take_calls": false,
        "coffee_price": "$3.50",
        "author_name": "John Doe"
    }
}
  • All Cafés: /all

  • Search Cafés by Location: /search_loc?loc=example_location

    Search for cafés by location.

    {
    "cafes": [
        {
            "id": 1,
            "name": "Example Café",
            "map_url": "https://example.com/map",

  • Search Cafés by Name: /search_name?name=example_cafe

    Search for cafés by name.

  • Search Cafés by Location (Unfound Result): /search_loc?loc=non_existent_location

    No cafés found for the provided location.

    {
    "cafes": {
        "error": {
            "Not Found": "Sorry, we don't have a cafe in that location."
        }
    }
}
